# Capstone Blog 

## Setup backend
1. `cd backend`
2. `npm install`
3. Copy `.env.example` -> `.env` and edit values if needed.
4. `npm start` (server at http://localhost:4000)

Tables are created automatically.

## Setup frontend
1. `cd frontend`
2. `npm install`
3. `npm start` (app at http://localhost:3000)

## Notes
- API base for frontend defaults to `http://localhost:4000/api`. Use `REACT_APP_API_BASE` to change.
- This is a learning/demo app; do not use in production without improvements.
